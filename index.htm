Caracteristicas Destacadas de HTML5

  1. Soporte multimedia nativa(audio y video).
  2. Elementos semanticos nuevos como < article>,< seccion>, y < nav>.
  3. Mejor soporte para graficos y juegos (canvas y SVG).
  4. Almacenamiento web local.
